Frontend para el simulador de núcleo atómico "Alkemist".

Por ahora no contiene gráficos. Se supone que el usuario controla un acelerador de partículas, el cual usa para disparar protones o neutrones para cambiar el núcleo atómico. El usuario así puede observar cómo un simple proton puede pasar a ser cualquier átomo.
